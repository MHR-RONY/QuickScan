## ✨ Features

- **7 content types** — URL, plain text, email, phone, SMS, WiFi, and vCard contact cards
- **50+ dot styles** — from classic squares and dots to hearts, stars, leaves, hexagons, sparkles, and more custom shapes
- **Custom corners** — mix and match border and center-eye styles, including leaf, octagon, notched, clover, and gear
- **Colors & gradients** — solid fills or linear/radial gradients, custom backgrounds, and transparency
- **Logos & badges** — upload your own image or pick from 55+ built-in icons and colored badges
- **Decorative frames** — coffee bag, shopping bag, polaroid, gift box, and scan labels
- **18 curated presets** — one-click themes like Neon Lime, Cyberpunk, Rose Gold, and Matrix
- **Randomize** — instantly generate a fresh style combination
- **Export anywhere** — download as PNG, SVG, or JPG at up to 800px
- **100% private** — everything runs client-side; your data never touches a server

## 📦 Getting Started

### Prerequisites

- [Bun](https://bun.sh) (recommended) or Node.js 20+

### Installation

```bash
# Clone the repository
git clone https://github.com/MHR-RONY/QuickScan.git
cd QuickScan

# Install dependencies
bun install

# Start the dev server
bun dev
```

The app runs at `http://localhost:3000`.

### Available Scripts

```bash
bun dev        # Start the development server
bun build      # Build for production
bun preview    # Preview the production build
bun lint       # Run ESLint
bun format     # Format with Prettier
```

## 🎨 How It Works

1. **Pick a content type** — choose what your QR code should link to
2. **Style it** — select a preset or fine-tune dots, corners, colors, and gradients
3. **Add a frame** — wrap your code in a decorative container
4. **Drop in a logo** — upload an image or choose a built-in icon
5. **Export** — download a pixel-perfect PNG, SVG, or JPG

> 💡 Always test-scan your QR code before printing at scale.
